# Task Manager (C++)

## Description
Task and project management system built with OOP in C++.

## Functionalities

1. Create project
2. Add task
3. Change task status
4. Add comments
5. Delete task
6. Filter tasks by status
7. Sort tasks by priority
8. Assign task to user
9. Show project progress
10. View task comments

## Run

Compile:

g++ main.cpp -o taskmanager

Run:

./taskmanager
